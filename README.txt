Input file description:

There are two examples: a network with 12 nodes and a network with 106 nodes.
For each example, there are 2 input files:
1. InputFile1.xlsx is the network topology and the data transmission frequency, data transmission amount, importance of each edge.
2. InputFile2.xlsx is the credibility and deviation of each node.

-------------------------------------------------------------------------------
How to run an example:

1. Run vva_install.exe to install the application vva.exe. 
(MATLAB Runtime 9.0 will be installed automatically if it is not on your computer.)
2. Run vva.exe, import InputFile1.xlsx and InputFile2.xlsx in turn according to the tips on the application GUI, and push the "Evaluate Credibility" button.
3. Wait for the end of the credibility evaluation.
